# CLAUDE

This repository includes a reference document for Claude-style assistance and project guidance.

## Use Case

When asking Claude for help, refer to `PROJECT_PLAN.md` to understand the current app goals, architecture, and MVP scope.

## What to check in `PROJECT_PLAN.md`

- Phase 1–5 feature progress and status
- Core data model and TypeScript types
- React Context + Hooks state management approach
- Tailwind CSS + dark mode requirements
- File structure and component responsibilities

## Best Practice

Align any recommendations or code changes with the plan in `PROJECT_PLAN.md`, especially the MVP scope and the decision to keep persistence client-side only.
